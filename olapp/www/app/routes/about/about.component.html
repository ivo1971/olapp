Deze applicatie is gemaakt met open-source software.
De volledige bron code kan je vinden op https://github.com/ivo1971/olapp/.

De client gebruikt:
  • Angular 2
  • Bootstrap
  • RxJs
  • Cordova

De websockets server gebruikt:
  • Boost
  • Curl
  • nlohmann Seasocks

De REST server - gehost op Heroku - gebruikt:
  • node js
  • npm
